## Instructions to start the test

### Using postman
- Run `npm install`
- Run `npm run dev`
- Open Postman and make a post request with <b>client_id</b>, <b>name</b> and <b>email</b>
- You will receive the requested response

### Locally
- Run `npm install`
- Run `npm run build`
- Run `npm run local`
- Check the file <b><i>stats.json</i></b> to see the data
